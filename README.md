# How to run

To run the app just run ./deploy.sh

This scripts installs all the dependencies of both apps and run each of them.

(The sh script maybe needs permissions, just run sudo chmod +x ./deploy.sh)

# Technologies used

## Front-end

- Vite
- ReactJS
- Tailwindcss
- Axios
- Typescript

## Back-end

- Typescript
- NodeJS
- Express
- MongoDB
